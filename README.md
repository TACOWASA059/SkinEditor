# SkinEditor
マインクラフトスキンの反転と頭の回転ができるプログラム

## installation
### 実行ファイル(.exe)を使う場合
/release/latest  
[リンク](https://github.com/TACOWASA059/SkinEditor/releases/)
からダウンロード
###  pythonから実行する場合  
####  requirements  
* windows
* python3.8.12
###  scripts
```bash
cd SkinEditor/
pip install -r requirements.txt
python3 app_plot.py
```
