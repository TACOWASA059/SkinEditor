# SkinEditor
マインクラフトスキンの反転と頭の回転などの一括処理ができるプログラム
![スクリーンショット 2022-10-14 011523](image/display.png)
## installation
### 実行ファイル(.exe)を使う場合
windowsのみ対応  
/release/latest  
[リンク](https://github.com/TACOWASA059/SkinEditor/releases/)
からapp_plot.exeをダウンロード  
そのまま実行
###  pythonから実行する場合  
####  requirements  
* python3.8.12
* tkinter
###  scripts
```bash
cd SkinEditor
pip install -r requirements.txt
cd scripts
python3 app_plot.py
```
